# MyChat
* 操作系统课设。

## 功能
* 多人聊天
* 区分 客户端 和 服务端
* 服务端支持踢人、封禁、敏感词过滤
* 支持消息弹幕
* 目前仅支持服务端和客户端在本机连接运行
  * 因此目前这仅是一个学习Qt用的小垃圾
  
## 代码
* c++/qml
* 使用Qt6/qml开发
